## **项目**

### [三高电子书站-网站链接：http://120.53.89.103:1024](http://120.53.89.103:1024)

` Spring Cloud`   `Spring Boot` `Nacos` `Docker` `Mysql` `RocketMQ` `Redis` `Elasticsearch` `Spring AI`  `XXL-JOB` `SkyWalking`

- **项目描述**：
  该项目是一个分布式在线小说阅读平台，包含门户、作家后台、管理后台等子系统。采用微服务架构，拆分为用户、书籍、搜索、网关、AI服务模块，提供小说阅读与创作服务。
- **工作内容**
  - 架构设计与容器化：基于 Spring Cloud Alibaba 微服务架构，独立完成用户、书籍、搜索等核心模块的后端设计与开发，并实现全链路服务的容器化部署与编排，搭建分布式系统基础。
  - 高性能混合检索：针对传统搜索语义理解弱的痛点，基于 Elasticsearch 设计 KNN 向量检索与 BM25文本检索的并行召回架构，引入 RRF 算法对结果重排序，提升“搜情节”等复杂查询的准确率。
  - AIGC 落地与监控：基于 Spring AI 接入大模型实现内容审核、润色及封面生成，设计长文本智能分片策略解决Token限制；定制 SkyWalking 链路追踪，监控AI调用端口，保障服务稳定性。
  - 缓存与消息处理：重构核心书籍发布链路，利用 RocketMQ 事务消息实现数据库与 ES 索引构建的异步解耦，将核心接口响应时间降低 72%（从274ms降至76ms）。
  - 数据读写优化：深度优化 Redis 交互，通过 Pipeline + Lua 脚本 将网络 IO 减少 93%，榜单查询性能提升 90%+。设计 Redis + Caffeine 多级缓存体系，配合 XXL-JOB 实现 Write-Behind 同步策略，将高频写操作转化为批量异步入库，解决高并发场景下的数据库写入瓶颈。
