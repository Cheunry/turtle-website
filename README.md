novel-cloud

├── novel-gateway -- 基于 Spring Cloud Gateway 构建的微服务网关
├── novel-search -- 基于 Elasticsearch 构建的搜索服务
├── novel-home -- 首页微服务
├── novel-book -- 小说微服务
├── novel-user -- 用户微服务
├──  novel-resource -- 资源微服务
└── novel-ai -- AI微服务
